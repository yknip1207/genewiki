
ConceptRecognizer: 
code that connects with the NCBO Annotator web service to identify ontology terms in the text of the gene wiki.  Includes code for downloading wiki pages, calling the annotator, and parsing the results.

genewikitools: 
a Google App Engine wep application used for viewing gene wiki statistics

referenceReport: 
code that parses the inline references out of wiki pages

SemanticWikiLinkSync: 
code used to mirror the gene wiki in Wikipedia with a gene wiki stored in a Semantic Media Wiki instance.  Along the way, semantics are added.

See project wiki for more information.
